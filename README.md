## Key Valt

Create the Keys in the Azure Keystore

for example:

az keyvault secret set --vault-name "FellowDEMOKeyVault" --name "DEMO" --value "yoursecret"

View your Key like

az keyvault secret show --name "AppSecret" --vault-name "<YourKeyVaultName>"
